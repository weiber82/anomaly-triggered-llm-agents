# anomaly-triggered-llm-agents

anomaly-triggered-llm-agents：基於異常觸發的盤中 LLM 多代理人決策系統
專案簡介

本專案旨在建立一套以「異常事件觸發（anomaly-triggered）」為核心的盤中金融市場分析與決策架構。
系統透過 L0 異常偵測 → L1 事件語義化 → L2 LLM 決策代理人 的分層式設計，
探討大型語言模型在短期市場事件中的解讀能力、行為模式與決策一致性。

此專案為論文研究之主要 Repository，包含文獻整理、研究設計、系統實作、實驗結果與完整紀錄。

研究目標

建立事件觸發式盤中市場分析流程。

使用 XGBoost 進行盤中異常偵測（L0）。

設計事件語義化與提示生成流程（L1）。

使用大型語言模型進行事件解讀與決策（L2）。

分析 LLM 在不同事件類型下的行為差異與推理特性。

建立可重現、可驗證、可擴充的盤中決策研究架構。

系統架構概述
L0 — 異常偵測（Anomaly Detection using XGBoost）

採用 XGBoost 進行監督式異常事件偵測

標註方式以未來短期報酬變化與波動度異常為基準

偵測結果作為後續 LLM 推理的觸發點

支援不平衡資料處理與 SHAP 模型解釋

L1 — 事件語義化與提示生成（Prompt Construction Layer）

將 L0 的事件轉成 LLM 可理解的語義描述

整合必要的市場上下文（如最近 1～5 分鐘的 K 線資料）

自動生成決策代理人之輸入格式

L2 — LLM 決策代理人（LLM Decision Agent）

由 LLM 對事件狀態進行推理與判斷

輸出格式包括：

多 / 空 / 不進場

推理理由

信心度（0–1）

研究重點為行為分析與一致性，而非最終報酬率
